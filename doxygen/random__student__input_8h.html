Detailed Description

Deklaruoja funkcijas, skirtas atsitiktinai generuoti studentų duomenis.

Šis failas apima funkcijos deklaraciją, kuri atsitiktinai generuoja studentų duomenis ir prideda juos į vektorių. Funkcija leidžia vartotojui įvesti norimą studentų skaičių, po kurio generuojami atsitiktiniai studentų vardai, pavardės, namų darbų balai (nuo 0 iki 10), namų darbų kiekis (nuo 1 iki 5), bei egzamino balas (nuo 0 iki 10).

Funkcija užtikrina, kad vartotojo įvestas studentų skaičius būtų teisingas (didesnis už 0). Jei įvestis neteisinga, vartotojas bus paprašytas įvesti ją dar kartą.

Failas naudoja klasę Studentas, kuri apibrėžia studento duomenis.

void addStudentsRandomly ( std::vector< Studentas > & studentai)

Atsitiktinai generuoja studentų duomenis ir prideda juos į pateiktą vektorių.

Ši funkcija prašo vartotojo įvesti norimą studentų skaičių, po kurio sugeneruoja:

  • Šabloninius studentų vardus ir pavardes (pvz., "Vardas1", "Pavarde1", ir t.t.),
  • Atsitiktinį namų darbų kiekį (nuo 1 iki 5) ir atsitiktinius namų darbų balus (nuo 0 iki 10),
  • Atsitiktinį egzamino rezultatą (nuo 0 iki 10).

Generuojant atsitiktinius skaičius, naudojama funkcija srand, kurios pradinis taškas nustatomas pagal dabartinį laiką, užtikrinant skirtingus rezultatus kiekvieną kartą.
